# Tracker II 射频组件架构澄清

## 核心架构结论

**YCC87 硅基 4 通道 DBF 芯片路线是 Tracker II 的正确答案**（2026-05-16 确认）

- 分立器件路线（每个天线单元配一个独立 T/R 芯片）已被否决
- YCC87 是硅基（非化合物）DBF 芯片，量产成本低，与数字集成电路兼容性好

## 阵面层级结构（2026-05-16 确认）

```
天线单元（1024阵元）
  ↓
YCC87 幅相控制芯片（4通道/片 × 256片）
  ↓
数字通道（64通道 → 16子阵，每子阵64阵元）
  ↓
子阵输出合并（16路数字中频）
```

## 关键组件说明

| 组件 | 技术路线 | 备注 |
|------|---------|------|
| 天线单元 | 瓦片式构型（Tile） | 2026-05-15 确认，区别于砖块式（Brick） |
| 幅相控制 | YCC87 硅基 4 通道 DBF 芯片 | 256 片，每片 4 通道 |
| 波控执行 | 分布式 ADDA + FPGA | 波形收发的执行端口 |
| 数字通道 | 64 数字通道 → 16 子阵 | Hybrid DBF：模拟子阵形成 + 数字波束形成 |
