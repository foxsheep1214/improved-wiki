# 文件恢复 — 误删 .md 文件从历史会话还原

## 触发条件

用户说"误删了"或"不小心删了"大硬件管理目录下的 .md 文件。

## 操作步骤

### 1. 确认文件清单

先用 `search_files` 在大硬件管理目录搜索 HTML 文件（可能被pandoc转换过）：

```
search_files(target="files", path="~/Documents/大硬件管理", pattern="*.html")
```

同时搜索大硬件管理目录内容：

```
search_files(target="files", path="~/Documents/大硬件管理", pattern="*")
```

### 2. 查询历史会话（session_search）

对每个待恢复文件，用 `session_search` 查询相关关键词：

```
session_search(limit=5, query="<filename> <相关人名/项目名>")
```

例如：
- `供应商管理.md` → `供应商管理.md 来访记录 俱吉 乾景 益丰`
- `产品及项目管理.md` → `产品及项目管理.md Tracker Tracer`
- `研发人力资源.md` → `研发人力资源.md 硬件设计部`
- `任务-研发资源.md` → `任务-研发资源.md`

### 3. 拼接还原

历史会话摘要通常包含：
- 文件的核心结构和表格内容
- 用户确认的决策结论
- 文件路径和关键命名

**局限性**（必须告知用户）：
- 对话摘要 ≠ 原文，内容有损压缩
- 部分字段会丢失（显示为"待补充"或"—"）
- 格式和措辞是AI重建，非原文
- **无 Time Machine 备份则无法100%恢复**

### 4. 写回文件

用 `write_file` 按还原的内容重建文件，路径放回原处：
```
~/Documents/大硬件管理/<文件名>.md
```
