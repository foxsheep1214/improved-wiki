# 砖块式 vs 瓦片式相控阵天线选型参考

> 适用场景：相控阵天线架构选型、系统方案讨论、供应商评估
> 基于知识：2026-05-15 讨论整理
> 关键词：砖块式/Brick、瓦片式/Tile、相控阵架构、T/R模块

---

## 核心区别

| 维度 | 砖块式（Brick） | 瓦片式（Tile） |
|------|----------------|----------------|
| **结构** | 每个T/R模块独立盒子封装 | T/R电路分层叠装（类似PCB积木），裸芯片直接贴装 |
| **单元密度** | 较高 | 受限于砖块厚度和层叠结构 |
| **成本** | 高（独立封装器件多） | 低（裸芯片批产成本远低于独立封装T/R模块） |
| **散热** | 差，分散热源难管理 | 好，可做大面积背板散热，适配GaN器件 |
| **可维护性** | 模块化，单砖可换 | 需拆整面，维护成本高 |
| **批产一致性** | 好控制 | 层压工艺要求高，一致性挑战大 |
| **适用频段** | Ka以上高密度小型化 | Ku/Ka大型地面/舰载 |
| **典型应用** | 导引头、数字波束形成（DBF）、多波束 | 地面警戒雷达、舰载防空反导雷达 |

---

## 选型决策树

```
需求分析
├── 小型化高密度？（导引头/机载）
│   └── 是 → 砖块式（Brick）
│       原因：单元间距小，独立封装T/R模块体积合适
│
├── 数字波束形成（DBF）/ 波形分集？
│   └── 是 → 砖块式
│       原因：每个单元需独立控制，砖块封装更灵活
│
├── 大型地面/舰载雷达 + 成本敏感？
│   └── 是 → 瓦片式（Tile）
│       原因：万级单元批产成本低，GaN散热设计优势
│
├── 高功率固态发射机（GaN器件）？
│   └── 是 → 瓦片式
│       原因：背板大面积散热，易管理热密度
│
└── 多波束/多功能（警戒+跟踪+成像）？
    └── 是 → 砖块式（架构更灵活）
```

## 工程实例

| 产品 | 架构 | 原因 |
|------|------|------|
| F-35 APG-81 | 砖块式 | 小型化、数字波束形成、多功能 |
| 地面警戒雷达（AN/MPQ-65） | 瓦片式 | 大型阵列、成本敏感、GaN高功率 |
| 舰载宙斯盾SPY-1 | 瓦片式 | 舰载空间有限、散热要求高 |
| 预警机雷达（E-3/E-767） | 砖块式 | 数字波束形成、远程预警 |
| 相控阵导引头 | 砖块式 | 体积极度受限、Ka频段 |

---

## 与供应链的关联

| 架构 | 关键器件 | 供应商类型 |
|------|---------|-----------|
| 砖块式 | 独立T/R模块封装（GaAs） | 化合物半导体IDM（中电科13所、55所） |
| 瓦片式 | 裸芯片T/R（GaN MMIC） | 晶圆代工 + 封装测试厂 |
| 瓦片式 | 高密度层压板 | 特殊工艺PCB厂（深南电路等） |

---

## 深圳塞防选型建议

针对Tracker II地面警戒雷达：
- **优先选瓦片式**：成本优先、地面固定平台散热条件好、批量生产
- **若未来需要数字多波束/认知雷达**：预留砖块式子阵升级空间
- **T/R组件选型**：关注瓦片式裸芯片供应商（国内：芯谷微、13所、55所）
