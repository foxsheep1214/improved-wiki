# 组内互审 SOP（同行评审门禁）

> 解决"无互审致低级错误频发"痛点。先在 Tracker II 试点，跑顺再推广。
> 原则：评审是**门禁**不是建议——未过不下单、不送样、不委外。结论记入任务表"评审"列。

## 触发点与门禁

| 触发点 | 必做评审 | 未过后果 |
|--------|---------|---------|
| 原理图冻结前 | 原理图 checklist 互审 | 不进 layout |
| layout 送样/委外前 | layout checklist 互审 | 不下单/不委外 |
| BOM 定版前 | BOM checklist 互审 | 不采购 |

## 互审人指派
- 由方向技术负责人（数字/射频/结构）指派 1 名**非作者**同行；跨方向板卡可加 1 名相关方向。
- 30 分钟以内、聚焦清单；发现项作者闭环后技术负责人确认通过。

## Checklist

**原理图（5–8 项）**
- [ ] 电源轨电压/电流余量、上下电时序
- [ ] 关键管脚连接/复位/启动 strap、上下拉
- [ ] 接口电平匹配、ESD/保护
- [ ] 时钟/差分/阻抗要求标注
- [ ] EMC/环境适应性（去耦、滤波、隔离）
- [ ] 器件选型：候选≥2、二供、出口管制核查
- [ ] 关键网络命名/标注清晰

**layout（5–8 项）**
- [ ] 叠层/阻抗与原理图要求一致
- [ ] 关键信号回流/参考平面完整、跨分割
- [ ] 电源平面载流/过孔/散热
- [ ] 差分等长、敏感线隔离、晶振包地
- [ ] 去耦电容就近、扇出
- [ ] DFM（间距/丝印/工艺角/拼板）
- [ ] 连接器/安装孔/结构干涉

**BOM（5 项）**
- [ ] 位号与原理图一一对应、无漏装/错装
- [ ] 封装与 footprint 匹配
- [ ] 关键器件二供/替代料、生命周期状态
- [ ] 出口管制/国产替代标注
- [ ] 价格/货期/MOQ 已确认

## 结论记录
- 任务表"评审"列填：`通过/有条件通过/不通过 + 评审人 + 日期`。
- 有条件通过：列整改项+闭环责任人，闭环后技术负责人销项。
